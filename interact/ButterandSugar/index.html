    김수빈,

    반효임,

    안해리,

    정민아,

    최선

    버터와 설탕은 홈디저트족과 입문자를 위한 음성 인식 기반 레시피 애플리케이션이다. 기존의 레시피 서비스와 달리 손을 사용하지 않아도 음성 인식 활성화를 통해 레시피 화면을 제어할 수 있으며 가로 모드의 화면을 제공하여 사용자가 편하게 요리를 진행할 수 있도록 도와준다.

    사용자의 요리 실력이나 취향을 고려한 맞춤 서비스와 똑똑한 레시피 도우미로서 재료 체크, 타이머, 데코 시뮬레이션 등을 통해 어렵고 복잡한 디저트 요리를 보다 쉽게 할 수 있도록 안내한다.
